Web Harmonium 和实体 Harmonium 并不是互相替代的关系。它们解决的是不同练习问题。

实体 harmonium 有气流、手感、簧片和真实共鸣；[Web Harmonium](/) 的优势是随时打开、标签清楚、便携、适合每天短时间重复。该选哪个，取决于你今天要练什么。

## Web Harmonium 的优势

最大的优势是快。打开浏览器，几秒钟就能开始。无需拉风箱、无需搬乐器、无需检查状态。

它很适合：

- 上课时快速确认 Sargam 音
- 唱歌前找到 Sa
- 在电脑前练一个短句
- 线上教学
- 在实体乐器前先构思旋律

对初学者来说，标签也是巨大帮助。直接在琴键上看到 Sargam 和西方音名，可以减少很多困惑。

## 实体 Harmonium 的优势

实体 harmonium 能训练软件无法提供的触感。你要控制风箱，感受琴键反馈，也能听到真实簧片随力度产生的变化。

如果你的目标是在实体乐器上表演，最终一定需要真实 harmonium。按键力度、气流控制、音栓、持续音和装饰音，在浏览器里都无法完全替代。

## 音色差异

Web Harmonium 如果有采样、簧片模式和混响，已经足够做音准、旋律、Sargam、声乐热身和创作草稿。

实体 harmonium 的声音更有层次。音色会受到风箱、房间、乐器品质和演奏方式影响。

你可以把网页版本当作练习笔记和音高参考，把实体乐器当作完整的演奏工具。

## 对声乐练习来说

对歌者来说，很多日常任务只需要稳定音高、快速移调和清晰音名。Web Harmonium 完全可以承担这些任务。

你可以先读 [寻找主音 Sa 指南](/blog/find-your-tonic-sa-on-web-harmonium)，再用 [声乐热身练习](/blog/vocal-warm-up-exercises-with-web-harmonium) 开始。

## 对 Raga 学习来说

早期 Raga 练习常常需要慢速重复、Sargam 辨认和短句记忆。Web Harmonium 很适合这些任务，因为键位和音名一直可见。

但当你开始练装饰音、触感和表现力时，实体 harmonium 或以声音为中心的练习会更重要。

## 对老师来说

老师可以把 Web Harmonium 当作共同参考。发一个链接，让学生切换到 Sargam 标签，然后练同一个短句，非常方便。

实体乐器在课堂里仍然重要，但网页版本能帮助学生在课后继续练习。

## 初学者需要马上买实体 Harmonium 吗？

如果你认真学习这件乐器，迟早需要。但在此之前，你完全可以先学习键位、音准、Sa 和基础 Sargam。

先用 [免费 Web Harmonium](/) 建立习惯。如果你能持续练习，并且想要更好的手感和音色，再考虑实体乐器。

## 总结

用 Web Harmonium 解决随时练、重复练、看得懂的问题；用实体 harmonium 解决触感、音色和长期演奏能力的问题。两者结合，会让练习更容易开始，也更容易深入。
