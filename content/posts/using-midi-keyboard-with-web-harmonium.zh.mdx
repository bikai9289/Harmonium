如果你有一台小型 MIDI 键盘，Web Harmonium 会变得更接近真实演奏。浏览器提供 harmonium 音色和标签，MIDI 键盘则给手指真实按键。

这个组合很适合 Sargam、声乐热身、Bhajan、Kirtan 和旋律草稿。先打开 [免费 Web Harmonium](/)，再按下面步骤连接。

## 你需要准备什么？

你需要：

- MIDI 键盘或控制器
- USB 线或可用的 MIDI 连接方式
- 支持 Web MIDI 的浏览器
- Play Harmonium 键盘页面

Chrome 或 Chromium 内核浏览器通常对 Web MIDI 支持更好。有些浏览器可能不会把 MIDI 设备开放给网页。

## 第一步：连接键盘

最好在打开 Web Harmonium 之前先连接 MIDI 键盘。这样浏览器更容易检测到设备。

如果键盘需要外接电源，先开机；如果是 USB 供电，插上后等几秒钟。

## 第二步：打开 Web Harmonium

打开 [Web Harmonium 键盘页面](/keyboard)。找到 MIDI 面板或输入设备选择。如果浏览器请求权限，允许 MIDI 访问。

在输入列表里选择你的键盘。如果看不到设备，可以刷新 MIDI 列表或重新加载页面。

## 第三步：测试几个音

在 MIDI 键盘上按几个键。你应该能听到浏览器里的 harmonium 音色。

如果没有声音，可以检查：

- 键盘是否已经开机
- USB 线是否正常
- 页面是否需要刷新
- 其他音乐软件是否占用了设备
- 当前浏览器是否支持 Web MIDI

## 第四步：选择标签

如果练印度音乐，可以切换到 Sargam 标签；如果要对照固定音高或其他键盘方法，可以使用西方音名。

实体 MIDI 按键和网页标签配合很好。手指能感受到键距，眼睛也能学习 Sa Re Ga Ma。

## 第五步：调整八度和移调

很多 MIDI 键盘比完整乐器小。如果音高太高或太低，可以用 Web Harmonium 的八度和移调功能调整。

对歌者来说，移调尤其重要。如果你在做声乐练习，可以先读 [如何找到主音 Sa](/blog/find-your-tonic-sa-on-web-harmonium)。

## 可以怎么练？

试试这些方法：

- 按住 Sa 并跟着唱
- 显示 Sargam 标签，慢慢弹 Sa Re Ga Ma
- 一句一句练 Bhajan 旋律
- 试弹一个 Raga 短句并重复三次
- 每轮练习后切换一次 Sargam 和西方音名

你也可以结合 [声乐热身指南](/blog/vocal-warm-up-exercises-with-web-harmonium) 使用。

## 关于浏览器权限

Web MIDI 是基于权限的。只有浏览器允许后，网站才能使用你的 MIDI 设备。如果以后不想开放，可以在浏览器设置里取消权限。

## 总结

MIDI 键盘能让 Web Harmonium 更有手感。你保留了浏览器的便利，同时获得了真实按键带来的练习体验。
